# 字符串

*字符串的定义*

~~~cpp
typedef struct{
    char ch[MAXSLEN+1];  //字符串的一维数组
    int length;          //表示字符串当前的位置
}SString;
~~~

*串的链式存储结构*

~~~cpp
typedef struct Chunk{
    char ch[CHUNKSIZE];
    struct Chunk *next;
}Chunk;

typedef struct{
    Chunk *head,*tail;   //串的头指针与尾指针
    int Curlen;          //串的当前长度
}LString;                //字符串的块状结构
~~~
